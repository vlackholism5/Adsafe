# AdSafe MVP 시나리오 (코드 기준 정리)

현재 구현된 코드 흐름에 맞춰 서류용으로 정리한 시나리오입니다.

---

## 3-1. AdSafe Mode (제작 모드) 시나리오

[목표]

👉 “이 문구, 위험한가?”를 제작 중에 즉시 인지

Flow

1. 사용자가 광고 문구 입력 (선택: 광고 유형, 문구 제목)
2. [검수하기] 버튼 클릭
3. 시스템 처리
   - 텍스트 전처리 (Normalization): 공백·특수기호·문장부호·숫자·대소문자 통일
   - 패턴 매칭 기반 룰엔진 실행 (키워드·정규식 + 이벤트 자동 체크 로직)
4. 결과 리포트 제공
   - 리스크 있음/없음 (요약 위험도: 신호 없음·낮음·보통·높음)
   - 리스크 계열 (각 항목별 riskCode·판정 단위)
   - 검수된 문구 영역에서 문제된 표현 하이라이트 표시
   - 상세 리스크 항목별: 매칭 표현, 왜 문제인지 설명, 수정 가이드
5. 수정 → 재검수 가능: 위 입력란을 수정한 뒤 [검수하기] 버튼을 다시 누르면 재검수
6. 검수 실행 시 검수 이력에 자동 저장 (검수 이력·검수 상세 페이지에서 조회 가능)

---

## 3-2. AduSafe Mode (교육 모드) 시나리오

[목표]

👉 “왜 이 표현이 문제인지” 스스로 판단할 수 있게 학습

Flow

1. AduSafe Mode 진입 후 ‘학습하기’ 또는 ‘복습하기’ 중 선택
   - 학습하기: 랜덤 10문제 새로 시작
   - 복습하기: 이전에 틀린 문제만 재풀이 (저장된 내용이 없으면 “복습할 내용이 없습니다” 안내)
2. 4지선다 랜덤 10문제 제시
   - 학습하기 선택 시 문제 은행에서 무작위 10문항 추출 후 1번부터 순서대로 표시
   - 각 문항: 문제 문장, 4개 선택지, [다음] 또는 [제출하고 결과 보기] 버튼
3. 즉시 피드백 (10문항 모두 풀이 후 결과 페이지에서 제공)
   - 문항별 정답/오답 표시
   - 선택한 답·정답·근거 설명·수정 가이드
4. 누적 학습 데이터 저장
   - 퀴즈 완료 시: 학습 이력 저장 (날짜, 정답 수, 총 문제 수, 정답률) → 학습 현황 페이지에서 조회
   - 틀린 문항이 있으면 복습용 데이터 저장 → 복습하기 선택 시 해당 문항만 재풀이 가능
