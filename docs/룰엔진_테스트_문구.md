# 룰엔진 검수 테스트 문구

AdSafe 검수 페이지(adsafe.html)에서 아래 문구를 **광고 문구**에 붙여넣고 [검수하기]를 눌러 각 리스크가 잘 잡히는지 확인하세요.

---

## 1. 과장·최상급 (RISK_SUPERLATIVE_*)

```
국내 최고 클리닉에서 최상의 결과를 약속합니다. 유일한 공인 시술법으로 완벽한 V라인을 만들어 드립니다.
```

**기대:** 절대적 최상급(최고, 최상, 유일, 완벽), 순위·공인 등

---

## 2. 효과보장·무위험 (RISK_GUARANTEE_*)

```
완치 보장합니다. 부작용 없음. 통증 없이 시술 후 당일 퇴원. 100% 만족 보장.
```

**기대:** 완치/해결 보장, 부작용 없음, 100%·확정성, 당일 퇴원 등

---

## 3. 치료기간 단정 (RISK_DURATION_*)

```
3일 완성, 일주일이면 끝. 간단하게 쉽게 해결. 바로 효과.
```

**기대:** 고정 기간(3일 완성, 일주일이면 끝), 단순화(간단하게, 쉽게), 즉시성(바로 효과) 등

---

## 4. 할인·이벤트 (RISK_INDUCEMENT_DISCOUNT, RISK_PRICE_EXCESSIVE)

```
신년 50% 할인 이벤트
턱보톡스 50%할인
10만원 -> 5만원
2026.01.01~2026.01.31
```

**기대:** 금전 혜택(할인, 이벤트), **50% 이상** → 과도 할인(고위험). 기간 있음 → "이벤트 기간 미표기"는 나오지 않아야 함.

---

## 5. 공포·외모 (RISK_FEAR_*, RISK_APPEARANCE_*)

```
방치하면 위험합니다. 지금 안 하면 늦습니다. 코끼리 다리 NO, 여신처럼 예뻐지세요.
```

**기대:** 위험 강조, 시급성, 신체 비하(코끼리 다리), 미화(여신) 등

---

## 6. 경험담·비교 (RISK_EXPERIENCE_*, RISK_COMPARISON_*)

```
실제 후기, 리얼 후기 만족. 다른 병원과 달리 우리만 가능한 시술. 아무데서나 받지 마세요.
```

**기대:** 환자 후기, 타기관 비교, 방식 우월, 비방 등

---

## 7. 보험·자격 (RISK_INSURANCE_*, RISK_QUALIFICATION_*)

```
실비보험 적용, 본인부담 없음. 국내 유일 전문병원. 명의가 직접 시술합니다.
```

**기대:** 보험 적용, 무료·환급 오인, 허위 자격(국내 유일), 임의 타이틀(명의) 등

---

## 8. 한 번에 여러 유형 (통합 테스트)

```
국내 최고 클리닉 완치 보장 부작용 없음
3일 완성 50% 할인 이벤트 2026.01.01~01.31
실제 후기 만족 다른 병원과 달리 실비 적용
```

**기대:** 과장, 효과보장, 무위험, 기간 단정, 50% 이상 할인, 경험담, 비교, 보험 등 여러 건 동시 적발.

---

## 9. 검수 통과용 수정 문구 (8번과 같은 맥락, 리스크 회피)

아래는 **8번 문구를 검수에 걸리지 않게** 바꾼 예시입니다. 의미는 비슷하게 두고, 규정에 걸리는 표현만 바꿨습니다.

```
턱 라인 시술 클리닉
증상 개선에 도움이 될 수 있습니다. 부작용이 적은 편이며, 의사와 상담 후 시술 가능합니다.
이벤트 기간: 2026.01.01~01.31. 할인율 30%. 적용 대상: 턱보톡스 1회.
많은 분들이 이용해 주셨습니다. 실비 보험 적용 여부는 상담 시 안내해 드립니다. (가입 조건에 따라 상이)
```

**수정 포인트**

| 원문 | 수정 | 회피한 룰 |
|------|------|------------|
| 국내 최고 클리닉 | 턱 라인 시술 클리닉 | 절대적 최상급(최고) |
| 완치 보장 부작용 없음 | 증상 개선에 도움이 될 수 있습니다. 부작용이 적은 편이며, 의사와 상담 후 시술 가능합니다. | 효과보장(완치 보장), 무위험(부작용 없음) |
| 3일 완성 50% 할인 이벤트 | 이벤트 기간: 2026.01.01~01.31. 할인율 30%. 적용 대상: 턱보톡스 1회. | 기간 단정(3일 완성), 50% 이상 할인 → 30%로 변경, 기간·조건 명시 |
| 실제 후기 만족 다른 병원과 달리 | 많은 분들이 이용해 주셨습니다. | 경험담(실제 후기), 타기관 비교(다른 병원과 달리) |
| 실비 적용 | 실비 보험 적용 여부는 상담 시 안내해 드립니다. (가입 조건에 따라 상이) | 보험 오인(무조건 적용) → 조건 명시 |

---

## 10. 리스크 거의 없음 (정상 문구)

```
턱 라인 개선 시술 상담 가능합니다.
이벤트 기간: 2026.01.01~01.31. 할인율 30%. 적용 대상: 턱보톡스 1회.
의사와 상담 후 결정하시면 됩니다.
```

**기대:** 30% 할인(50% 미만), 기간·조건 명시 → 과도 할인·기간 미표기 적발 없거나 최소. "금전 혜택" 안내만 나올 수 있음.

---

## 체크리스트

- [ ] 50% 할인 → "50% 이상" 고위험으로 잡히는지
- [ ] 40% 할인 → "50% 이상"으로 잡히지 **않는**지
- [ ] 기간(2026.01.01~01.31) 있으면 "이벤트 기간 미표기" 안 나오는지
- [ ] 입력에 없는 "최고", "부작용 없음" 등은 나오지 않는지
- [ ] 매칭 표현 하이라이트가 실제 문구와 일치하는지
